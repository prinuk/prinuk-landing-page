# פרינוּק - המכירה השבועית

דף נחיתה סטטי עבור המכירה השבועית של פרינוּק.

## פרסום ב-GitHub Pages

האתר בנוי כקבצים סטטיים בלבד:

- `index.html`
- `styles.css`
- `script.js`
- `assets/produce-photo-bg.png`

אחרי העלאה ל-GitHub:

1. נכנסים לריפו ב-GitHub.
2. פותחים `Settings`.
3. נכנסים ל-`Pages`.
4. בוחרים `Deploy from a branch`.
5. בוחרים את הענף `main` ואת התיקייה `/(root)`.
6. שומרים וממתינים לכתובת האתר שתופיע בעמוד.
